Machine Learning Vehicle Prediction Model (KNN)

Python

Matplotlib

NumPy

Sckit-learning

Gathered vehicle data such as horsepower, weight, top speed and acceleration time and stored data into an Excel spreadsheet. Used Jupyter Notebook along with Python to load the spreadsheet and create various tables for data visualization. Analyzed data from various tables and generated multiple plots using Matplotlib. Further developed a K-nearest-neighbors (KNN) machine learning prediction model to predict the make of a vehicle, given user provided input such as horsepower, weight, top speed and acceleration time parameters. Trained the model with the collected data and altered multiple parameters to increase prediction accuracy, resulting in a final accuracy value of 94.6%. Further looked into model accuracy using confusion matrices.

Food Bank Order System

Java

Swing

AWT

MySQL

JUnit

Created an application where users are able to generate a food hamper from a food bank. Contents of hampers were dictated based on user input, where users could enter values for family size (adult males, adult females, children). Based on this user input, a custom algorithm was created to assemble the most efficient food hamper while meeting the caloric and nutritional needs of all family members (for one week). Food inventory was provided in a live SQL database, and quantities of remaining food was updated after each hamper was created. Used the Swing and AWT framework to create the GUI, further generating 'orderform.txt' file that contained all neccesary information after a succesful order was placed. Developed the program based on a UML diagram, and tested all functionalities using JUnit Test Suite.
